Что такое HTML?

HTML (HyperText Markup Language) — это основной язык для создания веб-страниц. С его помощью можно структурировать текст, изображения, ссылки и прочие элементы на веб-страницах.

Основные возможности HTML

С помощью HTML можно создавать и форматировать документы, добавлять интерактивные элементы, такие как кнопки, формы и мультимедиа. В версии HTML5 добавлены новые теги и возможности.

HTML - это строительные блоки для создания веб-сайтов. Tim Berners-Lee.

Дополнительную информацию можно найти на странице Wikipedia о HTML.

Пример кода HTML:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Пример HTML</title>
</head>
<body>
    <h1>Это пример HTML структуры</h1>
    <p>Это простой текстовый абзац.</p>
</body>
</html>

Таблица HTML тегов:

Тег Описание
<p> Абзац текста
<h1> - <h6> Заголовки разных уровней
<a> Ссылка
